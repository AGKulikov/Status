# Функции автомобиля на HOME

Откройте **HOME → Настройки → Иконки, функции и приложения → Добавить иконку →
Функция автомобиля**. Приложение проверит ECARX и покажет доступные функции магнитолы.
Заполнять ID или коды вручную не требуется.

Для каждой плитки можно выбрать поведение нажатия:

- переключить Вкл/Выкл;
- перейти к следующему уровню или режиму;
- установить конкретный уровень, режим или температуру;
- отдельно назначить действие долгого нажатия.

В каталоге предусмотрены климат, AC и AUTO, передний/задний обогрев, скорость
вентилятора, температура водителя и пассажира, подогрев и вентиляция обоих передних
сидений, подогрев руля, режим движения, сервисное положение дворников, Auto Hold,
Start/Stop и экономия топлива. Неподдерживаемые конкретным автомобилем функции
отсеиваются по ответу ECARX.

Плитка получает текущее значение сразу после запуска HOME, затем подписывается на
изменения и дополнительно сверяет состояние каждые 30 секунд. Уровень или режим
показывается значком в углу; цвет иконки меняется вместе с состоянием. В оформлении
можно отключить цвета автомобиля и задать собственные активные/неактивные цвета.

Передний/задний обогрев и сервисное положение дворников представлены обычными
одноразовыми кнопками. Как и MConfig, приложение отправляет для них импульсную команду
`1` без предварительного чтения состояния. Дополнительных программных проверок скорости,
селектора и зажигания для кнопки дворников нет.

Управление багажником пока намеренно не добавлено: в проверенных исходниках нет
подтверждённого writable ID и зоны двери для этой модели. Подбирать такой ID наугад
небезопасно.

## Отдельная климатическая панель

В **HOME → Настройки → Настроить климатическую панель** доступна отдельная панель,
которая не зависит от пользовательских плиток. Она показывает подтверждённые ECARX
состояния питания, A/C, AUTO, температур обеих зон, вентилятора, подогрева и вентиляции
передних сидений, подогрева руля и обогрева стёкол. Неподдерживаемые конкретным
автомобилем функции скрываются после проверки каталога.

В редакторе можно без кода включать отдельные элементы, менять фон, прозрачность,
скругление, цвета и масштаб. Предпросмотр обновляется сразу и использует живые значения
автомобиля. Размер и позиция панели задаются в общем визуальном редакторе HOME в
пикселях и сохраняются вместе с остальной компоновкой.
