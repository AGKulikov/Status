# HA1076 release record

HA1076 is built from the published HA1075 / exact HA1068-derived launcher
baseline. It does not contain the experimental embedded navigator work.

## Android update identity

- Package: `ru.natro.statuswidget`
- Version name: `v2.8.2-ha1076`
- Version code: `208021076`
- Signing certificate SHA-256:
  `6e9855aedc008bbdd8a7fbf3f490be07f964b7ac658a837a1592647a08365c75`
- APK signing scheme: v3
- APK SHA-256:
  `64034ed0718fab5b6394c7a5cad8c04aac9402e5425814906d433713793bf9c2`
- APK size: `18003288` bytes

The package, certificate, and signing scheme match HA1075 exactly. The version
code is one higher than HA1075 (`208021075`), so Android accepts HA1076 as an
in-place update.

## Included fixes

- Replaced framework bitmap glyphs and launcher artwork with scalable vector
  resources, and request the highest available density for third-party app
  icons.
- Corrected cover/gate state semantics: closed is inactive; open, opening,
  closing, or stopped is active.
- Made media artwork freshness depend on image contents and track identity, so
  stale artwork is cleared and cannot return after a track change.
- Stabilized the status-row height before the first media metadata arrives.
- Preserved the Android 9 `ResolvableFuture` runtime dependency fix from
  HA1075.
