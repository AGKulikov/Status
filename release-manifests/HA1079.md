## Included changes

- Added one searchable, Apple-style settings centre with six clear groups,
  live summaries, responsive split/compact layouts and import/export/reset.
- Added a visible, working Back control to every detail Activity and kept it
  below both Android system insets and the app's own configurable status row.
- Removed the duplicate legacy Back control from Favorite Apps settings.
- Replaced remaining raw colour fields with the shared visual colour picker.
- Restored the configurable Information panel for vehicle, system and
  smart-home statuses with automatic icons and its own visual grid.
- Made the outdoor-temperature rolling median use only fresh measurements from
  the current sensor session and remain safe across Binder recovery.
- Preserved the vector/high-resolution icon, media freshness and no-jitter,
  smart-home state, four-corner panel resize and stable status-row-height fixes.
